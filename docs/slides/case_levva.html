05 / 12 Decisoes Tecnicas

5 ADRs Nygard

ADR-001

Bronze string-typed

Preserva qualquer formato original. Casts no Silver com try_cast ANSI-safe (Photon Spark 4.1 estrito).

ADR-002

SCD1 padrao + SCD2 demonstrativa

dim_cliente_history paralela. Hash MD5 sobre 4 colunas (segmento, UF, cidade, status). MERGE pattern Delta.

ADR-003

Medallion 3-camadas + landing

Schemas curtos (bronze/silver/gold/landing). Pattern canonico Databricks; queries mais limpas.

ADR-004

Free Edition vs Premium

Limitacoes conscientes (sem DLT/Workflows/RBAC/audit_log). Roadmap explicito para producao.

ADR-005

DQ flags + Quarantine pattern

Equivalente arquitetural a DLT no Free Edition. Bug raiz descoberto durante uplift: array_remove(arr, NULL) retorna NULL (NULL=NULL eh NULL); fix: usar array_compact().

06 / 12 Pente Fino DQ

Bug raiz descoberto e corrigido

Auditoria forense via SDD identificou 2 bugs DQ em direcoes opostas, ambos com mesma causa raiz.

Bug under-flag (silver_pedidos)

.otherwise(F.lit("warning"))  # rejected NUNCA

Antes: 0 rejected (artificial)

Depois: 24 rejected (real)

Bug over-flag (silver_ocorrencias)

F.lit(F.concat(...))  # forca non-null sempre

Antes: 270/270 rejected

Depois: 8 rejected, 193 warning, 69 clean

Causa raiz (descoberta via investigacao SQL)

SELECT array_remove(array(NULL, NULL, 'real'), NULL)
   -- retorna NULL (porque NULL = NULL eh NULL em SQL)
SELECT array_compact(array(NULL, NULL, 'real'))
   -- retorna ['real'] (correto)

Fix: substituir array_remove(arr, None) por array_compact(arr) em todos os silvers.
